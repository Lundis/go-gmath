# Benchmarking howto

To benchmark WASM, install wasmtime, then run

    cmd /c "set GOOS=wasip1&& set GOARCH=wasm&& go test -bench=. -exec wasmtime"

or

    GOOS=wasip1 GOARCH=wasm go test -bench=. -exec wasmtime
